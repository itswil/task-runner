Task Runners

What is a Task Runner?

Why use a Task Runner?

  • Simplifying workflow
  • Optimising assets (CSS/JS)
  • Maintaining code quality

How does it work?


# install node via nodejs.org

# using the node package manager, install grunt globally
npm install -g grunt-cli

# write your gruntfile.js

# start the watch task
grunt watch

CSS Preprocessors

Variables and Mixins


// _brand.scss
$facebook-blue: #3b5998;

@mixin box-shadow() {
  box-shadow: 10px 5px 5px darken($facebook-blue, 10%);
}

// _header.scss
.header--main {
  @include box-shadow;
  background-color: $facebook-blue;
}

CSS Preprocessors

Nesting


// _header.scss
.header--main {
  background-color: $facebook-blue;

  .text-logo {
    color: $facebook-white;
  }
}

// styles.css
.header--main {
  background-color: $facebook-blue;
}

.header--main .text-logo {
  color: $facebook-white;
}
