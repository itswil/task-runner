Task Runners

What is a Task Runner?

Why use a Task Runner?

  • Simplifying workflow
  • Optimising assets (CSS/JS)
  • Maintaining code quality

How does it work?


# install node via nodejs.org

# using the node package manager, install grunt globally
npm install -g grunt-cli

# write your gruntfile.js

# start the watch task
grunt watch

How do we use it at Macmillan?

  • Less
    • Convert to CSS
    • Concatenate
    • Minify
  • Uglify JS

What else can a Task Runner do?

  • Compress Images
  • Browsersync
  • Autoprefixer
  • UnCSS
  • Inline CSS/JS

There's ~200k more...

What are some alternatives?

  • Gulp (convention over configuration)
  • Brocoli
  • Mimosa
  • Brunch (extreme convention over configuration)
